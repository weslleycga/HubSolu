# 🚀 Tecnologias Utilizadas no Projeto

Este projeto (`hubsolu-frontend`) foi desenvolvido utilizando uma stack moderna e eficiente para aplicações web. Abaixo estão as principais tecnologias que compõem o projeto:

*   **[React](https://react.dev/) (`^18.2.0`)**: A biblioteca principal para a construção da interface de usuário (UI). Utilizada por sua componentização e reatividade.
*   **[Vite](https://vitejs.dev/) (`^5.2.11`)**: Ferramenta de build e servidor de desenvolvimento extremamente rápido, escolhido para substituir o Create React App ou Webpack, oferecendo melhor performance no desenvolvimento.
*   **[Tailwind CSS](https://tailwindcss.com/) (`^3.4.3`)**: Framework CSS utilitário (utility-first) utilizado para estilizar a aplicação de forma ágil, sem a necessidade de criar arquivos CSS separados para cada componente. Foram incluídos plugins extras como `@tailwindcss/forms` e `@tailwindcss/container-queries`.
*   **[React Router](https://reactrouter.com/) (`^6.23.1`)**: Biblioteca de roteamento padrão para aplicações React, permitindo a navegação entre diferentes páginas na aplicação Single Page Application (SPA).
*   **[Lucide React](https://lucide.dev/) (`^0.378.0`)**: Biblioteca de ícones open-source lindamente projetada, utilizada para ilustrar a UI.
*   **[Prettier](https://prettier.io/)**: Ferramenta de formatação de código para garantir um estilo padronizado em todo o projeto, integrada com um plugin do Tailwind para ordenar as classes css (`prettier-plugin-tailwindcss`).

# 🪜 Roteiro de Estudos (A Escadinha do Conhecimento)

Para compreender totalmente este projeto e conseguir modificá-lo ou criar algo parecido, é recomendado seguir uma ordem de estudos. Aqui está a "escadinha" que você deve subir:

### 🧱 Degrau 1: A Trindade da Web (Os Fundamentos)
*   **HTML**: Entender a estrutura da página, tags semânticas, formulários e acessibilidade.
*   **CSS**: Dominar layout (Flexbox e Grid), responsividade, box model e seletores.
*   **JavaScript (ES6+)**: Variáveis, funções (arrow functions), arrays, objetos, promises, async/await, map, filter e reduce. É o motor de tudo!

### 🎨 Degrau 2: Aceleração Visual com Tailwind CSS
*   Entender o conceito de *Utility-first CSS*.
*   Aprender como usar as classes do Tailwind para margens, cores, tipografia e flexbox.
*   Saber como o Tailwind ajuda na responsividade com prefixos como `md:`, `lg:`.

### ⚛️ Degrau 3: O Mundo React (Pensando em Componentes)
*   **JSX**: A sintaxe que mistura HTML dentro do JavaScript.
*   **Componentes**: Como quebrar a interface em partes reutilizáveis.
*   **Props**: Como passar informações de um componente "Pai" para um "Filho".
*   **State (Estado) & Hooks**: Como lidar com dados que mudam na tela (ex: `useState`, `useEffect`).

### 🗺️ Degrau 4: Navegação com React Router
*   Entender o que é uma SPA (Single Page Application).
*   Como criar rotas (`<BrowserRouter>`, `<Routes>`, `<Route>`).
*   Como navegar entre as páginas usando o `<Link>` ou o hook `useNavigate`.

### ⚡ Degrau 5: Ferramentas de Build (Vite)
*   Entender a diferença entre como um código React é escrito (JSX/ES6) e como o navegador entende (JS/HTML/CSS puro).
*   Aprender o que é o Vite e como ele agrupa e otimiza seus arquivos para produção.

### 🛠️ Degrau 6: Boas Práticas (Prettier)
*   Entender a importância de manter um padrão visual de código.
*   Configurar ferramentas no VS Code (ou editor de preferência) para formatar o código ao salvar.

### 🏆 Degrau 7: Prática no Projeto Hubsolu
*   Navegue pelos arquivos que já existem no projeto.
*   Identifique como cada "degrau" anterior está sendo aplicado na prática.
*   Faça pequenas alterações, salve e veja a mágica acontecer!
